# cafanasyev-sapient-proto

Compiled Python classes (with type stubs) for the SAPIENT
[BSI Flex 335 v2.0](https://github.com/dstl/SAPIENT-Proto-Files) protobuf
message definitions.

```python
from sapient_msg.bsi_flex_335_v2_0.sapient_message_pb2 import SapientMessage
```

Type stubs (`.pyi`, generated by `protoc --pyi_out`) ship in the
wheel, so IDE autocomplete and `mypy --strict` work out of the box. For
idiomatic dataclasses, see the companion distribution
`cafanasyev-sapient-betterproto`.

See the [repository](https://github.com/cafanasyev/py-sapient-proto) for
details and licensing (proto definitions are Dstl (c) Crown Copyright 2024,
implementing BSI Flex 335 © The British Standards Institution).
